# أبو ليلة ERP - نظام إدارة الموارد

نظام متكامل لإدارة الموارد والمخزون والمبيعات والموارد البشرية مصمم خصيصاً لشركات التصنيع والملابس.

## المميزات الرئيسية

### 1. **المخازن (Inventory)**
- تعريف وإدارة الأصناف والمنتجات
- إدارة المنتجات النهائية والخامات
- تتبع حركات المخزون
- التحويلات بين المخازن
- سحب الخامات للتصنيع
- الجرد التسويتي

### 2. **نقطة البيع (POS)**
- واجهة بيع سريعة مع ماسح باركود
- إدارة سلة المشتريات
- خصومات وطرق دفع متعددة
- طباعة الفواتير بصيغ مختلفة (80mm، A4، A5)

### 3. **المشتريات والموردين**
- إدارة قائمة الموردين
- أوامر الشراء
- فواتير الشراء
- مردودات الشراء
- مدفوعات الموردين

### 4. **المبيعات**
- فواتير المبيعات
- مردودات المبيعات
- إدارة العملاء والحدود الائتمانية

### 5. **العملاء (CRM)**
- قائمة العملاء النقدي والآجل
- كشوفات الحساب
- تحصيلات العملاء
- تتبع أرصدة العملاء

### 6. **المالية**
- إدارة الخزائن والبنوك
- تسجيل المصروفات
- الإيرادات الأخرى
- إدارة الشيكات
- التحويلات المالية

### 7. **الموارد البشرية**
- إدارة الموظفين
- تسجيل الحضور والانصراف
- السلف والخصومات
- بنود الإنتاج (الأجر بالقطعة)
- تسجيل الإنتاج اليومي
- حساب الرواتب

### 8. **التقاریر**
- تقاریر المخازن
- تقاریر المبيعات
- التقاریر المالية
- تقاریر الموارد البشرية

### 9. **الإعدادات والصلاحيات**
- إدارة المستخدمين
- الصلاحيات والأدوار
- إعدادات النظام

## التكنولوجيا المستخدمة

- **Framework**: Next.js 15 (App Router)
- **State Management**: Redux Toolkit + React-Redux
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **Deployment**: Vercel

## البدء السريع

### المتطلبات
- Node.js 18+ و npm/pnpm

### التثبيت

```bash
# تثبيت المكتبات
pnpm install

# تشغيل النظام في وضع التطوير
pnpm dev

# إنشاء نسخة الإنتاج
pnpm build
pnpm start
```

النظام سيعمل على `http://localhost:3000`

## ملاحظات مهمة

- البيانات يتم حفظها في الحالة (State) على الجانب العميل فقط - في الإنتاج قد تحتاج إلى ربط مع قاعدة بيانات
- جميع الصفحات مترجمة بالكامل إلى اللغة العربية
- النظام يدعم الاتجاه من اليمين إلى اليسار (RTL)
- لا يوجد حد لعدد المنتجات أو الموظفين أو الفواتير
